# 九州南北朝ヘックス戦記 (Kyushu Nanbokucho Hex Wars)

## 概要
南北朝時代の九州を舞台にした、ヘックス（六角形）マップ採用の戦略シミュレーションゲームです。
懐良親王や菊池武光といった南朝の英雄、そして名将・今川了俊率いる北朝軍の戦いを再現します。

## 登場人物
- **懐良親王:** 征西大将軍。南朝軍の象徴。
- **菊池武光:** 肥後の名将。高い攻撃力を誇る南朝の柱石。
- **今川了俊:** 北朝の九州探題。知略と防御に長けた名将。
- **一色範氏:** 初期九州探題。
- **少弐頼尚:** 九州独自の有力勢力。

## ゲームシステム
- **ヘックスマップ:** 六角形のマス目で構成された九州の戦場。
- **ターン制:** 南朝と北朝が交互に行動。
- **戦闘:** 隣接する敵ユニットを攻撃。HPが0になるとユニットは消滅。
- **技術スタック:** Rust (Tauri) + React (TypeScript)
